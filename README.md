   <code>cd QueryScheduler </code>

   <code>mkdir build-QueryScheduler</code>  
4. run cmake on the project directory:  
   <code>cmake ..</code>  
5. run make:  
   <code>make </code>
6. modify the file <code>name.txt</code> with your favourite text editor, and add your classmate name
